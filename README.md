# One Memories

A collaborative wedding-film atelier for the San Francisco Bay Area.

**Your idea, our work.**

## Deploy on Vercel

1. Push this repo to GitHub
2. Connect to Vercel at https://vercel.com/dashboard
3. Vercel auto-deploys on every push

## Local Dev

```bash
npm install
npm run dev
```

Open http://localhost:3000
